# talkie-generator

See [examples](./examples)

## install

```
$ npm i -g talkie-generator
```

## usage

### init

```
$ talkie init <outputDirPath>
```

`<outputDirPath>: {String} Default: 'templates'`

### generate

```
$ talkie <templatesDirPath>
``` 

*required*

`<templatesDirPath>: {String}`

## structure

- template.ejs
- config.js
- index.md
